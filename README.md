# Habit Manager

Este proyecto es un gestor de hábitos que permite a los usuarios agregar, eliminar y listar sus hábitos de manera sencilla y eficiente.

## Estructura del Proyecto

- **CMakeLists.txt**: Archivo de configuración principal para CMake.
- **README.md**: Documentación del proyecto.
- **include/habit_manager.h**: Declaración de la clase `HabitManager`.
- **src/main.cpp**: Punto de entrada de la aplicación.

## Instrucciones de Instalación

1. Clona el repositorio:
   ```
   git clone <URL_DEL_REPOSITORIO>
   ```
2. Navega al directorio del proyecto:
   ```
   cd habit-manager
   ```
3. Crea un directorio de construcción:
   ```
   mkdir build && cd build
   ```
4. Ejecuta CMake para configurar el proyecto:
   ```
   cmake ..
   ```
5. Compila el proyecto:
   ```
   make
   ```

## Uso

Una vez compilado, puedes ejecutar el gestor de hábitos con el siguiente comando:
```
./habit-manager
```

Sigue las instrucciones en pantalla para gestionar tus hábitos.
